## Project Overview

This project focuses on analyzing **Blinkit grocery data** using **Microsoft Power BI** to uncover insights related to sales performance, product categories, and customer purchasing patterns. The dashboard transforms raw grocery sales data into clear and actionable business insights.

## Objectives

* Analyze overall grocery sales and revenue
* Identify top-selling grocery categories and products
* Understand customer buying behavior
* Track key performance indicators (KPIs) through interactive visuals

## Dataset Description

The dataset contains grocery-related information such as:

* Product categories
* Sales values
* Order details
* Time-based data

## Tools & Technologies

* **Microsoft Power BI**
* **Power Query** for data cleaning and transformation
* **DAX** for calculated measures
* **Excel / CSV** grocery dataset

## Dashboard Features

* Total grocery sales overview
* Category-wise and product-wise analysis
* Sales trends over time
* Interactive slicers and filters for detailed exploration

**Note:**
GitHub does not preview `.pbix` files.
Please download the file and open it using **Microsoft Power BI Desktop**.

## How to Use

1. Clone or download the repository
2. Open `blinkit.pbix` in Power BI Desktop
3. Use filters and visuals to explore grocery sales insights
